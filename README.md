# RTS_Game

Developed with Unreal Engine 4


Movement Controls

  1. MoveForward
    *. W
    *. UpArrow
  2. MoveBackwards
    * S
    * DownArrow
  3. MoveRight
    * D
    * RightArrow
  4. MoveLeft
    * A
    * LeftArrow
  
  5. ZoomIn
    * MouseWheelUp
    * PageUp
  6. ZoomOut
    * MouseWheelDown
    * Page Down
  7. ZoomReset
    * MiddleMouseButton + Shift
    * Home
  
  8. Pan
    * MiddleMouseButton
  9. PanReset
    * MiddleMouseButton + Alt
    * Home + Alt
  10. SpeedModifier
    * LeftShift
    * RightShift
